# 📚 Système de Gestion de Bibliothèque Universitaire

**Université Polytechnique de Bingerville - L3 MIAGE**  
**Année 2024-2025**





## 🏗️ Architecture
📱 Mobile (Flutter)  ←→  🖥️ Backend (Spring Boot)  ←→  🗄️ PostgreSQL
🌐 Web (Flutter)     ←→
## 📂 Structure
/backend          → API REST Spring Boot
/frontend-mobile  → App mobile Flutter
/frontend-web     → App web Flutter
/database         → Scripts SQL
/docs             → Documentation
## 🚀 Installation

### Backend
```bash
cd backend
./mvnw spring-boot:run
```

### Mobile
```bash
cd frontend-mobile
flutter run
```

### Web
```bash
cd frontend-web
flutter run -d chrome
```

## 🔧 Prérequis

- Java 17+
- Maven 3.8+
- PostreSQL
- Flutter 3.16+
